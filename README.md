# Learning backend with django

## React Frontend Todo

A full-featured Todo application built with React, connected to a Django REST API backend. 

## Required Backend Repository

[django-backend-todo](https://github.com/ajay-nishad25/django-backend-todo)

## Getting Started

1. Clone this repository
2. Open a terminal inside the `react-frontend-todo` folder
3. Run the following command:
```bash
npm install
```

The dashboard includes the following features:

1. **Todo List** — View all your todos at a glance
2. **Global Search** — Search across all todos instantly
3. **Pagination** — Displays total pages; each page contains up to 12 todo items
4. **Page Navigation** — Jump directly to a specific page by entering a page number
5. **View Mode Toggle** — Switch between card view and list view
6. **Filter Options** — Filter todos by status or other criteria
7. **Create New Todo** — Add a new todo item
